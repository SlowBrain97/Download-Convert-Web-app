# ダウンローダー Web アプリケーション

## 概要
このプロジェクトは、動画や音楽のダウンロードと変換を行うWebアプリケーションのフロントエンド部分です。

## 主な機能
- **動画ダウンロード** - YouTube、Twitter、Instagramなど主要プラットフォームからの動画ダウンロードをサポート
- **音楽抽出** - 動画から高音質な音楽ファイルを抽出可能
- **形式変換** - MP4, MP3, WAV, AVI など様々な形式への変換に対応
- **プレイリスト対応** - 複数動画の一括ダウンロードと変換
- **進捗表示** - ダウンロードや変換の進捗状況をリアルタイムで表示
- **レスポンシブデザイン** - スマートフォンやタブレットなど、あらゆるデバイスで快適に利用可能

## 技術スタック

### フロントエンド
- **React 18** - インタラクティブなUI構築のためのJavaScriptライブラリ
- **TypeScript** - 型安全なコード開発を実現
- **Vite** - 高速な開発サーバーとビルドツール
- **Tailwind CSS** - ユーティリティファーストなCSSフレームワーク
- **React Query** - サーバー状態管理とデータフェッチング
- **Axios** - HTTPクライアントライブラリ
- **React Router** - シングルページアプリケーションのルーティング
- **Framer Motion** - スムーズなアニメーション実装

## 環境変数

プロジェクトを実行するには、以下の環境変数を設定する必要があります：

```env
VITE_API_BASE_URL=バックエンドAPIのベースURL
VITE_APP_NAME=アプリケーション名
VITE_APP_VERSION=アプリケーションバージョン
```

## セットアップ方法

### 必要な環境
- Node.js (v16以上推奨)
- npm または yarn

### インストール
```bash
# 依存パッケージをインストール
npm install
# または
yarn install
```

## 開発サーバーの起動
```bash
npm run dev
# または
yarn dev
```

## ビルド
```bash
npm run build
# または
yarn build
```
